Dependency Vulnerability Checker: Developed in my final year at Technological University Dublin, in partnership with Bank of America, this web application offers a robust solution for cybersecurity and dependency management. It features secure user authentication and a file upload portal for dependency files. The app scans these files for vulnerabilities, offering solutions for identified issues. Each analysis is saved to the user’s account, allowing for ongoing monitoring and updates on new vulnerabilities. Additionally, the app includes a cyber security feature for scanning files and URLs for malware, adding another layer of protection. This project represents a synergy of academic knowledge and real-world application, emphasizing proactive cybersecurity in software development.
